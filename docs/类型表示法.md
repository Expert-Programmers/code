类型定义可以使用以下的非正式语法:

T1 :: A | B | C ...

它的意思是T1被定义为A或B或C之一.

用这种表示法,可以定义一下Eralng类型如下:

Type :: any() | none() | pid() | port() | reference() | [] |

Atom() | binary() | float() | Fun() | Integer | [Type] |

Tuple | Union | UserDefined

Union :: Type1 | Type2 ...

Atom :: atom() || Erlang_Atom

Integer :: integer() | Min .. Max

Fun :: fun() | fun((...) -> Type)

Tuple :: tuple() | {T1, T2, ... Tn}
